# In class

## What we will do

1. Introduce the course and one another.
2. Distinguish AI, machine learning, deep learning, and generative AI.
3. Investigate conflicting airline guidance in [The Broken Oracle](../../activities/session-1/broken-oracle.md).
4. Use live polls to distinguish fixed automation, predictive AI, generative assistance, and an agentic workflow.
5. Compare a chatbot with an agent and identify what enters an LLM's current context.
6. Learn the roles of Git, GitHub, commits, branches, forks, pushes, and pull requests.
7. Complete the [course agent lab](../../labs/session1-build-an-agent/GUIDE.md).

The class deck is a separate course material. Use this page to find the work and required output.

## Leave with

- One evidence-based diagnosis and controlled repair from The Broken Oracle.
- An `agent-check.md` file inside your own `student-work/<github-login>/` path.
- A reviewed draft pull request, or a precise record of the setup blocker and next recovery step.

Inspect every changed path and the complete diff. Course files and other students' folders are read-only, and anything pushed to a fork or pull request is public.
