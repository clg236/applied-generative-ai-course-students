# Applied Generative AI · student materials

This repository contains the course sessions currently released to students. It is read-only course reference material.

Use it beside your private course workspace:

```text
courses/
  applied-generative-ai-course-students/   # read-only materials
  applied-generative-ai-work/              # your private work
```

Pull this repository at the start of each class to receive newly enabled sessions. Do not place your API key or coursework here.

Your OpenCode agent runs from `applied-generative-ai-work`, reads these materials, and writes only to the private workspace. Follow the setup instructions in `setup/setup-guide.md` and the current session guide.
