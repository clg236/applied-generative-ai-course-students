# Student work

Student pull requests use a GitHub-login namespace:

```text
student-work/<github-login>/session-work/
student-work/<github-login>/submissions/
```

Everything here is public. Do not add real names when a GitHub handle is sufficient. Never add grades, feedback, attendance, accommodations, student identifiers, email addresses, credentials, private messages, learning profiles, proprietary records, or identifiable field media.

The pull-request check requires every changed path in a student contribution to remain inside the pull-request author's folder.
